# Bilidown Mac 使用说明

这是一个 macOS 版 B 站视频下载工作流。给它一个 Bilibili 链接，它会下载并合并成 MP4 文件。

## 1. 快速开始

在当前项目根目录执行：

```bash
./mac-bilidown/bin/bilidown doctor
```

如果检查通过，就可以下载：

```bash
./mac-bilidown/bin/bilidown download "https://www.bilibili.com/video/BV..."
```

默认下载到：

```text
~/Downloads/Bilidown
```

下载完成后，命令行会打印最终 MP4 文件路径。

## 2. 常用命令

查看环境是否正常：

```bash
./mac-bilidown/bin/bilidown doctor
```

下载单个视频：

```bash
./mac-bilidown/bin/bilidown download "https://www.bilibili.com/video/BV..."
```

下载到指定目录：

```bash
./mac-bilidown/bin/bilidown download "https://www.bilibili.com/video/BV..." --output "$HOME/Desktop/Bilibili"
```

查看视频信息：

```bash
./mac-bilidown/bin/bilidown info "https://www.bilibili.com/video/BV..."
```

输出 JSON，方便给其他工具调用：

```bash
./mac-bilidown/bin/bilidown info "https://www.bilibili.com/video/BV..." --json
```

## 3. 画质选择

默认使用最佳画质：

```bash
./mac-bilidown/bin/bilidown download "https://www.bilibili.com/video/BV..." --quality best
```

指定 1080P 以内：

```bash
./mac-bilidown/bin/bilidown download "https://www.bilibili.com/video/BV..." --quality 1080p
```

指定 720P 以内：

```bash
./mac-bilidown/bin/bilidown download "https://www.bilibili.com/video/BV..." --quality 720p
```

只下载音频：

```bash
./mac-bilidown/bin/bilidown download "https://www.bilibili.com/video/BV..." --quality audio
```

视频模式会优先选择更通用的 H.264/AAC MP4，拿不到时再回退到 yt-dlp 能获取的最佳格式。

## 4. 登录和 Cookie

默认会读取本机 Chrome 的 B 站登录 Cookie：

```bash
./mac-bilidown/bin/bilidown download "https://www.bilibili.com/video/BV..."
```

这样可以使用你 Chrome 里已经登录的账号权限，例如 1080P、会员画质或受限视频。Cookie 只在本机读取，不会上传到这个项目或第三方服务。

首次使用时，macOS 可能会弹出 Keychain 授权提示。请在 Terminal 里执行一次下载命令，并允许访问 Chrome Safe Storage。

如果只想下载公开视频，不读取 Chrome Cookie：

```bash
./mac-bilidown/bin/bilidown download "https://www.bilibili.com/video/BV..." --no-cookies
```

如果 Keychain 授权弹窗无法显示，默认会在 20 秒后失败并返回 exit code `20`。可以延长等待时间：

```bash
BILIDOWN_COOKIE_TIMEOUT=60 ./mac-bilidown/bin/bilidown download "https://www.bilibili.com/video/BV..."
```

## 5. 多 P 和合集

默认只下载当前视频：

```bash
./mac-bilidown/bin/bilidown download "https://www.bilibili.com/video/BV..."
```

如果要下载整个多 P 或合集：

```bash
./mac-bilidown/bin/bilidown download "https://www.bilibili.com/video/BV..." --playlist all
```

## 6. 给其他工具调用

这个工具适合作为工作流被 Raycast、快捷指令、脚本或其他自动化工具调用。

最简单的调用方式：

```bash
/Users/shishen/my_work/copy_bilibili_videodownload/mac-bilidown/bin/bilidown download "$URL"
```

获取机器可读的视频信息：

```bash
/Users/shishen/my_work/copy_bilibili_videodownload/mac-bilidown/bin/bilidown info "$URL" --json
```

公开视频下载，不触发 Chrome Cookie：

```bash
/Users/shishen/my_work/copy_bilibili_videodownload/mac-bilidown/bin/bilidown download "$URL" --no-cookies
```

## 7. 依赖文件

项目已经内置了 macOS 依赖：

```text
mac-bilidown/vendor/darwin-arm64/
mac-bilidown/vendor/darwin-x64/
```

如果以后需要重新下载或修复依赖：

```bash
./mac-bilidown/scripts/fetch-vendor-deps.zsh
```

同时补齐 Apple Silicon 和 Intel Mac：

```bash
./mac-bilidown/scripts/fetch-vendor-deps.zsh --all-arch
```

如果其他工作流需要 ffprobe：

```bash
./mac-bilidown/scripts/fetch-vendor-deps.zsh --all-arch --with-ffprobe
```

校验依赖：

```bash
cd mac-bilidown
shasum -a 256 -c vendor/checksums.sha256
```

## 8. Exit Codes

| Code | 含义 |
| --- | --- |
| `0` | 成功 |
| `2` | 参数错误或 URL 不是 B 站链接 |
| `10` | 内置依赖缺失或不可执行 |
| `20` | Chrome Cookie 或 Keychain 访问失败 |
| `30` | 下载、解析或合并失败 |

## 9. 常见问题

### 提示 Chrome Keychain access timed out

说明当前环境无法完成 Chrome Cookie 解密授权。解决方式：

```bash
./mac-bilidown/bin/bilidown download "https://www.bilibili.com/video/BV..."
```

请在 macOS Terminal 里运行，并允许 Keychain 弹窗。若不需要登录画质，可以加：

```bash
--no-cookies
```

### 想确认工具是否正常

运行：

```bash
./mac-bilidown/bin/bilidown doctor
```

### 下载结果不是预期画质

先确认 Chrome 已登录 B 站，并且该账号有对应画质权限。然后尝试：

```bash
./mac-bilidown/bin/bilidown download "https://www.bilibili.com/video/BV..." --quality best
```

### 想只测试，不下载视频

运行：

```bash
./mac-bilidown/bin/bilidown info "https://www.bilibili.com/video/BV..." --no-cookies
```

或者：

```bash
./mac-bilidown/bin/bilidown info "https://www.bilibili.com/video/BV..." --json --no-cookies
```
